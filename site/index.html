نسخهٔ نمونه

فهرست دروس

برای هر درس یک پوشه ایجاد شده است. روی لینک‌ها کلیک کنید:

پزشکی داخلی

site/courses/medicine/

دندانپزشکی

site/courses/dentistry/

نمونه فصل: مقدمات (Ch01)

site/courses/medicine/residency/pre-intern/ch01/
نکته: از لینک‌های relative استفاده کنید. مثال: ../ch02/

راهنمای سریع Deploy با GitHub Pages

  1. محتوی سایت را در پوشهٔ site/ نگه دارید.
  2. فایل workflow در .github/workflows/pages.yml است.
  3. در تنظیمات ریپازیتوری، Pages را روی GitHub Actions قرار دهید.
تهیه شده جهت نمونهٔ ساختار سایت آموزشی
